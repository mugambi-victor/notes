My first angular app

  1. install node js.
  1. Install Angular CLI
npm install -g @angular/cli
  1. Create a workspace and create your new angular app with this command:
ng new my-app
  1. navigate to the project folder and start the development server.
ng serve --open
  1. The project structure:
  • src: Holds the source code for your application, including components, services, modules and routing configurations.
  • node_modules: Contains downloaded dependencies for your project.
  • package.json: lists project dependencies and scripts.
  • angular.json: holds various Angular configuration options.
Creating your first component.

Components are the building blocks of angular projects.

To create a component run:

ng generate component <name_of_component>
//example
ng generate component ProductsComponent
  • This generates the component files (TypeScript, HTML, CSS) in the src/app directory.
  • The files look like this:
  1. typescript file(products-component.component.ts
import { Component } from '@angular/core';

@Component({
  selector: 'app-products-component',
  standalone: true,
  imports: [],
  templateUrl: './products-component.component.html',
  styleUrl: './products-component.component.css'
})
export class ProductsComponentComponent {

}
  1. The Html file(products-component.component.html
<p>Lorem ipsum dolor sit, amet consectetur adipisicing elit. Dolore aut cupiditate esse tempora ipsum excepturi maxime hic, voluptatem laudantium atque enim nihil necessitatibus numquam sit debitis magni rem blanditiis autem!</p>

the html doesnt need the upper structure as its alredy there in the index.html file which comes with the angular app. the rest of the code goes inside this index.html root element.

When the component is created, you may need to include it in the UI to displayed to the user. to do this, we need to include the component inside the app.component.ts.

import { CUSTOM_ELEMENTS_SCHEMA, Component } from '@angular/core';
import { RouterOutlet } from '@angular/router';
// name of the export class in component's
// Typescript file 
import { ProductsComponentComponent } from './products-component/products-component.component';


@Component({
  selector: 'app-root',
  standalone: true,
// import the class here once more
  imports: [RouterOutlet, ProductsComponentComponent],
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
 name = 'victor';

}
